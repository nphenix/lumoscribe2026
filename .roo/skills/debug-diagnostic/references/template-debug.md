## 环境信息字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 操作系统 | 运行平台 | Windows 11, Linux Ubuntu |

| 复现步骤 | 重现问题的步骤 | 1. 启动服务器... |

## 根因分析方法

| 步骤 | 说明 |
|------|------|
| 假设 | 提出可能的根因 |
| 验证 | 设计验证方法 |
| 测试 | 执行验证方法 |
| 结论 | 确定最终根因 |

## 代码变更类型

| 类型 | 说明 |
|------|------|
| 新增 | 添加新文件或代码 |
| 修改 | 变更现有代码 |
| 删除 | 移除代码 |

## 注意事项

- 复现步骤必须可执行
- 假设要有针对性
- 验证方法要可操作
- 修复后要验证
- 经验教训要总结
